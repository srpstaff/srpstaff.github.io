# ER:LC Moderation Command Guidelines

Admin commands are powerful tools designed to maintain order, not to be abused or used for personal entertainment.

### General Command Rules
* **Minimum Letters:** When using targeting commands (e.g., `:kick`, `:ban`, `:jail`), you must type at least **4+ letters** of the player's username to prevent targeting the wrong user.
* **On-Duty Requirement:** Commands must only be executed while actively on-duty, unless handling an immediate emergency or clearing a glitch upon joining.
* **No Retaliation:** Using commands out of anger, frustration, or personal retaliation will result in an immediate administrative strike.

### Permitted Moderation Actions
* `:m` / `:h` – Used strictly for official server announcements or warnings.
* `:warn [user] [reason]` – Official logging warning for minor rule infractions.
* `:kick [user] [reason]` – For persistent disruption after verbal warnings.
* `:jail [user]` – For temporary holding during an active sit or investigation.
